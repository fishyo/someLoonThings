## 🛠 手动配置指南

### 1. 插件安装

- **自动**：点击上方的“点击添加”链接。
- **手动**：进入 Loon 的 `插件` 页面 -> 点击右上角 `+` -> `从链接安装` -> 输入对应的 `.lpx` 或 `.plugin` 原始链接。

### 2. 脚本使用 (节点信息)

- 插件安装后，在 **节点列表** 中 **长按** 任意节点。
- 在弹出的菜单中选择 **脚本 (Script)**。
- 点击 **节点信息** 即可查看该节点的网络详细信息。
