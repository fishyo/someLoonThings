# My Scripts

个人自动化脚本集合

## 📁 目录结构

### Ninebot (九号出行)
- `9bot.js` - 签到脚本 (支持自动重试)
- `9bot-set-cookie.js` - Cookie 获取脚本
- `ninebot/` - 相关脚本目录
  - `ninebot_cookie.js` - Cookie 管理
  - `ninebotsign.js` - 签到实现

## 🚀 使用方法

### 九号出行签到

1. 首先运行 Cookie 获取脚本获取授权信息
2. Cookie 会自动保存到持久化存储
3. 配置定时任务运行 `9bot.js` 进行每日签到

## ⚙️ 功能特性

- ✅ 自动重试机制 (最多3次)
- ✅ 网络超时保护 (15秒)
- ✅ 智能错误处理
- ✅ Cookie 失效检测
- ✅ 通知推送支持
